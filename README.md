**The bot is currently using the following models:**
Reasoning: DeepSeek-V4-Flash
Image Recognition: Qwen-Vl-Max
Image Generation: Qwen-Image-2.0-pro

**Commands of the bot:**
| # - for chat

| #learn - to learn new knowledge
| #learn that - learn last searched answer
| #learn from <url>  - learn from a link

| #knowledge - to check knowledge base
| #forget n - forget the n-th knowledge
| #clean data base - to clean the local knowledge base

| #thinking - turning on and off thinking

| #search - to search online
| #autosearch - check if autosearch is on or off
| #autosearch on/off - turn on and off the autosearch
 *autosearch is off initially

| #imagine ... - create image
| #edit ... - upload an image and edit
| #describe ... - for describing an image

**Character roleplay:**
It roleplays as the character Cyrene from Honkai Star Rail.
